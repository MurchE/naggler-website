Terms

The short version.

Naggler is free software we're glad you're using. Use it for whatever you want, except don't use it to break the law or hurt other people. It comes with no warranty — if it eats your task list, we'll feel bad but we won't pay for it.

Last updated: April 19, 2026

What this covers

These terms apply when you download the Naggler desktop app, use naggler.com, or talk to the Naggler Cloud AI relay. If you're using your own API key or a local Ollama model, Anthropic's or Ollama's terms also apply to that piece.

The software itself

  • Naggler is distributed as a pre-built binary for Windows, macOS, and Linux. Source is on GitHub.
  • You can install it, run it, uninstall it. You can copy the binary to another computer. You can decompile it if you enjoy that sort of thing.
  • You can't repackage Naggler and sell it as your own product. You can't strip out the feedback email or pretend the app is someone else's work.
  • The "Naggler" and "Naggy" names and the hedgehog character belong to us. The ideas and the code are yours to learn from.

Naggler Cloud (the AI relay)

  • We pay for the compute so new users can try Naggy without setting up an API key. There's a per-install daily rate limit (currently 100 interactions/day) so one heavy user can't empty our pot for everyone else.
  • If you hit the limit, the app tells you and offers to switch you to your own key or a local model. You're never blocked from using Naggler — only from using our free credits.
  • We reserve the right to cap, pause, or remove Naggler Cloud if the economics stop working or someone is clearly abusing it. If we do, we'll tell you in the app, not quietly.

Your data

See the privacy page. Short version: your task data lives on your own computer, we don't have it, we can't hand it over.

No warranty

Naggler is provided "as is." We don't promise it won't crash, won't lose your data, or won't be wrong about whether you should have emailed Omer back. If something goes sideways, we're sorry — we'll probably want to hear about it at feedback@naggler.com — but we aren't liable for damages.

Changes

When we change these terms in a way that actually matters, we'll update this page and tell people who signed up for the download email. Typo fixes and small clarifications won't get a notification.
